How to update this page

All content lives in one place: the DATA object near the top of this file's <script>. Edit the text there, not the markup. To add an item, copy an existing entry in the list and change its fields.

  1. Edit project/index.html in the agustin-8thLight/portfolio repo.
  2. For an opportunity, set status to open, next, doing, or done, and fill owner.
  3. Update LAST_UPDATED at the top, commit, and push. Pages redeploys in about a minute.
